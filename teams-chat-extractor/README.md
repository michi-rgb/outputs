# Teams Chat Extractor

Microsoft Teams ブラウザ版のチャット画面から、**ユーザー名・日時・チャット内容**を自動抽出するChrome拡張機能です。

---

## インストール方法

1. `teams-chat-extractor` フォルダをわかりやすい場所に保存
2. Chrome で `chrome://extensions` を開く
3. 右上の「**デベロッパーモード**」をオンにする
4. 「**パッケージ化されていない拡張機能を読み込む**」をクリック
5. `teams-chat-extractor` フォルダを選択

---

## 使い方

1. ブラウザで [https://teams.microsoft.com](https://teams.microsoft.com) を開く
2. 抽出したいチャット画面に移動する
3. ツールバーの拡張機能アイコン（💬）をクリック
4. 出力形式を選択（CSV / テキスト / TSV / JSON）
5. 「**チャットを抽出**」ボタンをクリック
6. 「クリップボードにコピー」または「ファイルに保存」で取り出す

---

## 出力形式

| 形式 | 説明 | おすすめ用途 |
|---|---|---|
| CSV (UTF-8 BOM付き) | カンマ区切り | Excel・スプレッドシート |
| テキスト | 読みやすい整形 | ドキュメント・レポート |
| TSV | タブ区切り | データ処理 |
| JSON | 構造化データ | 開発・データ活用 |

### CSVサンプル
```csv
timestamp,author,body
"2025-01-15 10:30:45","山田 太郎","おはようございます。本日の会議ですが..."
"2025-01-15 10:31:02","鈴木 花子","よろしくお願いします。"
```

## セレクターが合わない場合（Teamsのアップデート後など）

TeamsはDOMを頻繁に変更するため、セレクターがズレることがあります。

### 確認方法

1. Teams のチャット画面で `F12`（開発者ツール）を開く
2. メッセージ要素を右クリック →「検証」でDOMを確認
3. `content.js` の `SELECTORS` 定義を実際のクラス名・属性に合わせて更新

### よくある修正箇所 (`content.js`)

```javascript
const SELECTORS = {
  messageItem: [
    '[data-tid="message-body"]',   // ← ここのセレクターを更新
    ...
  ],
  author: [
    '[data-tid="message-author-name"]',  // ← 著者名要素
    ...
  ],
  ...
};
```

---

## 注意事項

- **ブラウザ版 Teams 専用**です（デスクトップアプリは対応外）
- APIキーや認証情報は一切使用しません
- データはローカルで処理され、外部に送信されません
- 画面に表示されているメッセージのみ抽出されます（スクロールして読み込んだ分も含む）
- TeamsのDOMアップデートにより動作しなくなる場合があります

## トラブルシューティング

| 症状 | 対処 |
|---|---|
| 「0件」と表示される | チャット画面が開いているか確認。ページをリロードして再試行 |
| ユーザー名が空欄 | Teamsのアップデートでセレクターが変化した可能性。`content.js`の`SELECTORS.author`を確認 |
| 日時が取得できない | 同上。`SELECTORS.timestamp`を確認 |
| 拡張機能が表示されない | `chrome://extensions` でエラーがないか確認 |
