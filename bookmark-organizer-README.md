# ブックマーク自動整理アプリ — README

`bookmark-organizer.html` をブラウザで開くだけで動作する、単一ファイルのWebアプリです。

---

## 処理の流れ

```
① ブックマークHTMLエクスポート（ユーザー操作）
        ↓
② ファイル読み込み（ブラウザ内）
        ↓
③ メタデータクリーニング（ブラウザ内）
        ↓
④ AI分類リクエスト（外部APIへ送信）
        ↓
⑤ レスポンス検証・欠落チェック（ブラウザ内）
        ↓
⑥ 整理済みHTMLをダウンロード
```

### 各ステップの詳細

**① ブックマークのエクスポート**

Chrome / Edge のブックマークマネージャ（`Ctrl + Shift + O`）→ 右上メニュー →「ブックマークをエクスポート」で `bookmarks.html` を保存します。実行前に必ずバックアップしてください。

**② ファイル読み込み**

エクスポートした HTML ファイルをドラッグ＆ドロップ、またはクリックして選択します。ファイルはブラウザ内のメモリにのみ読み込まれ、外部には送信されません。

**③ メタデータクリーニング**

AI に送る前に、不要な属性を自動削除してトークン数を削減します。

| 削除される属性 | 内容 |
|---|---|
| `ADD_DATE` | 追加日時のタイムスタンプ |
| `LAST_MODIFIED` | 最終更新日時 |
| `ICON` / `ICON_URI` | ファビコンデータ（data:URI は特に大きい） |
| `LAST_CHARSET` | 文字コード情報 |

削減率は通常 **40〜80%** で、API コストと処理時間を大幅に抑えられます。

**④ AI分類リクエスト**

クリーニング済みのブックマーク HTML をプロンプトに含め、AI API へ送信します。

| プロバイダー | エンドポイント | デフォルトモデル |
|---|---|---|
| Mistral AI | `api.mistral.ai/v1/chat/completions` | `mistral-small-latest` |
| Gemini (Google) | `generativelanguage.googleapis.com/v1beta/...` | `gemini-2.5-pro` |

AI への指示（プロンプト）の要点:
- 全ブックマークを **10〜15 カテゴリ**に分類
- フォルダ名は日本語 + 絵文字
- 階層は **1階層のみ**（ネストしない）
- ブックマークを一つも削除・統合しない
- Netscape Bookmark File Format の完全な HTML で出力

**⑤ レスポンス検証・欠落チェック**

AI からのレスポンスを受け取った後、以下を自動チェックします。

- Markdown コードフェンス（` ```html ` など）の除去
- `NETSCAPE-Bookmark` または `<DL` が含まれるか確認
- 元ファイルの全 URL と出力 HTML の全 URL を比較し、**欠落したブックマークを特定**
- 欠落があれば一覧表示し、CSV でのダウンロードも可能

**⑥ ダウンロード・インポート**

「整理済み bookmarks.html をダウンロード」ボタンで保存します。その後：

1. ブックマークマネージャを開く
2. 既存のブックマークを削除（事前バックアップ必須）
3. 「ブックマークをインポート」でダウンロードしたファイルを選択

## APIキーの取得

**Mistral AI（推奨・無料枠あり）**
- [console.mistral.ai/api-keys/](https://console.mistral.ai/api-keys/) でアカウント作成 → APIキー発行
- 無料枠: 月 1B トークン、2 RPM（リクエスト/分）
- ブックマーク数が多い場合はレート制限に注意

**Gemini (Google)**
- [aistudio.google.com/apikey](https://aistudio.google.com/apikey) で無料取得
- `gemini-2.5-pro` → `gemini-2.5-pro-preview-06-05` → `gemini-1.5-pro` の順にフォールバック

APIキーは入力後 **ブラウザの localStorage に保存**されます（次回起動時に自動入力）。

---

## セキュリティについて

- APIキーと bookmarks.html の内容は **ブラウザ内でのみ処理**されます
- 送信先は選択した AI プロバイダーの公式 API のみです
- このアプリ自体はサーバーを持たず、外部にデータを送りません
